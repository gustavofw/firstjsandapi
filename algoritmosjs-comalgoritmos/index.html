JavaScript

  • É uma linguagem de programação.
  • Interpretada pelo navegador para o desenvolvimento de lógica de programação do front-end de uma aplicação Web.
  • Pode ser utilizada no também no back-end através do NodeJS.
  • É fracamente tipada, mas pode ser fortemente tipada quando for utilizada junto com o TypeScript.
  • Case sensitive.
  • Sua sintaxe e baseada na linguagem C.
  • É extremamente poderosa para maniputação do DOM (Document Object Mode).
